READ ME 

ECG.ino : 
	Brancher la carte Arduino The Things Network puis par l'intermediaire de l'Arduino IDE : televerser le programme sur la carte 
	
ecg_generator.ino : 
	Brancher la carte Arduino Uno puis par l'intermediaire de l'Arduino IDE : televerser le programme sur la carte 
	
localInterfaceNodeRed.json : 
	Importer ce flux sur le Node Red heberge sur la Raspberry Pi : menu -> import -> paste flow json or select a file to import 

remoteInterfaceNodeRed.json : 
	Importer ce flux sur le Node Red heberge sur le serveur distant : menu -> import -> paste flow json or select a file to import
